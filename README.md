# VLC Pro Translator 🎬🌍

**VLC Media Player with Auto Language Detection & Real-time Translation for Android**

[![Build APK](https://github.com/YOUR_USERNAME/vlc-pro-translator/actions/workflows/build-apk.yml/badge.svg)](https://github.com/YOUR_USERNAME/vlc-pro-translator/actions/workflows/build-apk.yml)

## ✨ Features

- 🎤 **Real-time Speech Recognition** - Converts audio to text instantly
- 🔍 **Auto Language Detection** - Automatically detects video language
- 🌍 **Real-time Translation** - Translates to 19 languages instantly
- 🎯 **Smart Caching** - No duplicate translations
- 📱 **VLC-style Interface** - Familiar orange theme
- 💯 **Free & Open Source** - No ads, no tracking

## 📲 Download APK

1. Click **"Actions"** tab above ⬆️
2. Click the latest workflow run
3. Scroll down to **"Artifacts"** section
4. Download **"VLC-Pro-Debug-APK"** or **"VLC-Pro-Release-APK"**
5. Install on your Android phone!

## 🚀 How to Use

1. Install the APK on your Android phone
2. Open the app
3. Tap **"Open File"** and select a video
4. Tap **📝 Caption button** to start speech recognition
5. Tap **🌐 Translate button** to enable translation
6. Select your **target language** in Settings (⚙️)
7. Enjoy auto-translated captions!

## 🌐 Supported Languages

**Auto-detects:** English, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Korean, Chinese, Hindi, Arabic, Dutch, Polish, Turkish, Vietnamese, Thai, Indonesian, Malay

**Translates to:** Any of the above!

## 📋 Requirements

- Android 5.0+ (API 21)
- Internet connection (for translation)
- Microphone permission (for speech recognition)

## 📝 Notes

- Speech recognition works **offline** on most Android devices
- Translation requires **internet** (uses free MyMemory API)
- Free API limit: ~1000 words/day (sufficient for movies)
